</b>

<b>연결 문제를 겪는다면:</b>
- 최근 앱 화면: Mi Bandage 고정 (pull down the app and click on lock icon)
- 폰 배터리 설정/배터리 최적화: Mi Bandage(을)를 최적화하지 않도록 설정
- settings/theme/notification shade 활성화

<b>문제가 지속된다면:</b>
- 폰을 다시 시작
- 제게 이메일 작성

<b>주요 기능:</b>
- 배터리 상태, 남은 예상 시간 표시
- 발신자와 일반 및 인터넷 수신 전화 신호 표시
- 밴드/어메이즈핏에 발신자와 부재중 전화 신호 표시
- 일반 및 인터넷 수신 전화 거절, 응답 및 음소거
- 자동 문자 변환
- 밴드에 앱의 알림 텍스트 표시
- 흔한 이모티콘 표시

<b>미밴드/어메이즈핏 연결 기능:</b>
- 음량 변경
- 밴드 진동
- 기기 화면 깨우기
- 알림을 받았을 때 즉각적인 신호 보내기
- 다시 연결할 때 놓친 알림 표시
- 알림 리마인더
- 화면이 꺼져있을 때에만 신호 보내기
- ongoing notifications are enabled or disabled

<b>밴드 연결 해제 기능:</b>
- icon displaying and vibrate on Band
- playing ringtone on device, vibration, screen wake up
- 음량 변경
- 기기 모니터링 agent: if someone tampered with the device while you are away, you will know (wake up the screen, power on, motion): notification on the device and Weibo icon on the Band

<b>알림 모니터링:</b>
- unique signal configuration for apps notification
- displays the app's notification texts on Band
- pre icon/vibration to have more time to watch the Band

<b>터치 모니터링:</b>
- 전화 응답, 스피커폰 (turn off the launch music app when headset is connected setting, otherwise the music starts) (not all devices are supported)
- 전화 거절, ending (not all devices are supported)
- call out
- toggle call mute
- 내 기기 찾기
- repeat the phone's notifications on the bracelet
- control music player (play, pause, next, previous, volume up/down, show music track info)
- start timer/stopwatch/interval, showing remaining/elapsed time
- 사진 촬영
- 앱 런처
- Tasker task 런처

<b>어메이즈핏 Bip/Cor:</b>
- watch face change
 
<b>걸음수 모니터링:</b>
- daily/weekly/monthly charts: displays the measured values
- more goals can be set

<b>수면 모니터링:</b>
- daily/weekly charts: displays the measured values
 
<b>심박수 모니터링:</b>
- 설정 가능: monitoring starting signal, show pulse, low and high pulse signal, start/end time, repeat interval
- 현재 심박수 표시
- 일간/주간/월간 차트

<b>시간 모니터링:</b>
- 폰 알람이 울릴 때 자동으로 신호 보내기
- 알람 설정
- 타이머: 알람 종류, 리마인더, remaining time display on Band
- 스톱워치: 알람 종류, 리마인더, 경과된 시간, lap time recording, lap time displaying on Band
- 주기: 간격 연습을 위한 타이머

<b>분실 모니터링:</b>
- Band signal: when specified period of time expires Band will alert you
- device signals: when specified period of time expires, device send signals repeatedly if not connected to the bracelet. This makes it easier to find the device.

<b>Agent 기록:</b> 
- view events detected by the device monitoring agent
